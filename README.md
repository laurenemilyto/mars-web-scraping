## About This Project
This project scrapes a news site using BeautifulSoup, Pandas, and Requests/Splinter. MongoDB was then used with Flask to display results on an HTML page.

## Technologies & Methods
- BeautifulSoup
- Pandas
- Requests/Splinter
- MongoDB 
- Flask
- Bootstrap
