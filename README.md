## About This Project
This project is designed to build web application that scrapes various websites for data related to the Mission to Mars and displays the information in a single HTML page. 

## Technologies & Methods
- BeautifulSoup
- Pandas
- Requests/Splinter
- MongoDB 
- Flask
- Bootstrap
